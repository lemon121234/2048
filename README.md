# 2048 遊戲

一個使用純 HTML、CSS 和 JavaScript 實現的 2048 小遊戲，適合面試展示使用。

## 📸 截圖預覽

遊戲採用現代深色主題設計，具有流暢的動畫效果。

## 🎮 遊戲規則

1. 使用 **方向鍵** (↑↓←→) 或 **WASD** 來移動所有方塊
2. 當兩個相同數字的方塊相撞時，它們會 **合併** 成一個
3. 每次移動後，會在空白處隨機出現一個新的方塊（2 或 4）
4. 目標是合併出 **2048** 這個數字！
5. 當沒有空位且無法移動時，遊戲結束

## ✨ 功能特色

- 🎨 **現代化 UI** - 深色主題、漸層色彩、流暢動畫
- 📱 **響應式設計** - 支援桌面和手機瀏覽器
- 👆 **觸控支援** - 支援滑動手勢操作
- 💾 **最高分記錄** - 使用 LocalStorage 保存最高分
- 🔄 **新遊戲** - 隨時可以重新開始

### 使用技術

- **HTML5** - 語義化標籤
- **CSS3** - Flexbox、Grid、動畫、漸層
- **JavaScript ES6+** - Class、箭頭函數、模板字串
- **LocalStorage** - 本地儲存最高分

## 🚀 如何運行

1. 下載或克隆此專案
2. 直接用瀏覽器打開 `index.html` 即可遊玩

```bash
# 或使用任何本地伺服器
npx serve .
```
